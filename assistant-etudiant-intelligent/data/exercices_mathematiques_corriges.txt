SÉRIE 1 : CALCUL DIFFÉRENTIEL

Exercice 1 : Calcul de dérivées
Calculer les dérivées des fonctions suivantes :

a) f(x) = x³ + 2x² - 5x + 1
Solution : f'(x) = 3x² + 4x - 5

b) g(x) = sin(x² + 1)
Solution : g'(x) = cos(x² + 1) × 2x = 2x cos(x² + 1)

c) h(x) = ln(x² + 3x)
Solution : h'(x) = (2x + 3)/(x² + 3x)

d) k(x) = e^(2x+1)
Solution : k'(x) = 2e^(2x+1)

Exercice 2 : Étude de fonction
Étudier la fonction f(x) = x³ - 3x² + 2

Solution :
1. Domaine : R
2. Dérivée : f'(x) = 3x² - 6x = 3x(x - 2)
3. Points critiques : x = 0 et x = 2
4. Signe de f'(x) :
   - x < 0 : f'(x) > 0 (croissante)
   - 0 < x < 2 : f'(x) < 0 (décroissante)
   - x > 2 : f'(x) > 0 (croissante)
5. Extrema : maximum en x = 0 (f(0) = 2), minimum en x = 2 (f(2) = -2)

Exercice 3 : Équations différentielles
Résoudre les équations différentielles suivantes :

a) dy/dx + 2y = 4x
Solution : y = 2x - 1 + Ce^(-2x)

b) d²y/dx² + 4y = 0
Solution : y = A cos(2x) + B sin(2x)

c) dy/dx = y/x (x > 0)
Solution : y = Cx

SÉRIE 2 : ALGÈBRE LINÉAIRE

Exercice 4 : Calcul de déterminants
Calculer les déterminants suivants :

a) det([[2,3],[1,4]])
Solution : 2×4 - 3×1 = 8 - 3 = 5

b) det([[1,2,3],[0,4,5],[0,0,6]])
Solution : 1×4×6 = 24 (matrice triangulaire)

c) det([[1,1,1],[1,2,3],[1,3,6]])
Solution : 1×(2×6-3×3) - 1×(1×6-3×1) + 1×(1×3-2×1) = 1×3 - 1×3 + 1×1 = 1

Exercice 5 : Systèmes linéaires
Résoudre les systèmes suivants :

a) x + 2y = 5
   3x - y = 1
Solution : x = 1, y = 2

b) 2x + y - z = 1
   x - y + z = 2
   x + 2y + z = 0
Solution : x = 1, y = -1, z = 2

Exercice 6 : Valeurs propres
Trouver les valeurs propres de A = [[3,1],[0,2]]

Solution :
det(A - λI) = det([[3-λ,1],[0,2-λ]]) = (3-λ)(2-λ) = 0
Donc λ₁ = 3 et λ₂ = 2

SÉRIE 3 : INTÉGRALES

Exercice 7 : Calcul d'intégrales
Calculer les intégrales suivantes :

a) ∫[0→π] sin x dx
Solution : [-cos x][0→π] = -cos π + cos 0 = 1 + 1 = 2

b) ∫[0→1] x² dx
Solution : [x³/3][0→1] = 1/3

c) ∫[1→e] (1/x) dx
Solution : [ln x][1→e] = ln e - ln 1 = 1 - 0 = 1

d) ∫[0→∞] e^(-x) dx
Solution : [-e^(-x)][0→∞] = 0 - (-1) = 1

Exercice 8 : Intégration par parties
Calculer ∫[0→1] x e^x dx

Solution :
u = x, dv = e^x dx
du = dx, v = e^x
∫ x e^x dx = x e^x - ∫ e^x dx = x e^x - e^x = e^x(x - 1)
[0→1] : e¹(1-1) - e⁰(0-1) = 0 - (-1) = 1

SÉRIE 4 : PROBLÈMES APPLIQUÉS

Exercice 9 : Optimisation
Un rectangle a un périmètre de 20 cm. Quelles sont les dimensions pour maximiser l'aire ?

Solution :
Soit x et y les dimensions : 2x + 2y = 20, donc y = 10 - x
Aire : A = xy = x(10 - x) = 10x - x²
A' = 10 - 2x = 0 → x = 5
Donc x = 5 cm et y = 5 cm (carré)

Exercice 10 : Modélisation
La population d'une ville croît selon P(t) = P₀ e^(0.02t) où t est en années.
Si P₀ = 10000, quand la population atteindra-t-elle 20000 ?

Solution :
20000 = 10000 e^(0.02t)
2 = e^(0.02t)
ln 2 = 0.02t
t = ln 2 / 0.02 ≈ 34.7 années

MÉTHODES DE RÉSOLUTION

1. Dérivation :
   - Règles de base (puissance, produit, quotient, composée)

2. Intégration :

   - Équations à variables séparables
   - Équations linéaires du premier ordre
   - Équations du second ordre à coefficients constants

FORMULES DE RÉFÉRENCE

Dérivées :
- (x^n)' = n x^(n-1)
- (sin x)' = cos x
- (cos x)' = -sin x
- (e^x)' = e^x
- (ln x)' = 1/x

Intégrales :
- ∫ x^n dx = x^(n+1)/(n+1) + C
- ∫ sin x dx = -cos x + C
- ∫ cos x dx = sin x + C
- ∫ e^x dx = e^x + C
- ∫ 1/x dx = ln |x| + C

Déterminants :
- 2×2 : det([[a,b],[c,d]]) = ad - bc
- 3×3 : règle de Sarrus
