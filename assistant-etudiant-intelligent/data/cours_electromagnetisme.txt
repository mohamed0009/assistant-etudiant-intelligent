COURS D'ÉLECTROMAGNÉTISME - LICENCE 2
PHYSIQUE GÉNÉRALE

CHAPITRE 1 : ÉLECTROSTATIQUE

1.1 LOI DE COULOMB
La force électrostatique entre deux charges ponctuelles :
F⃗ = k(q₁q₂/r²)r̂
où k = 1/(4πε₀) = 9×10⁹ N·m²/C²

1.2 CHAMP ÉLECTRIQUE
Définition : E⃗ = F⃗/q
Champ d'une charge ponctuelle : E⃗ = k(q/r²)r̂

Principe de superposition : E⃗_total = ΣE⃗_i

1.3 POTENTIEL ÉLECTRIQUE
Définition : V = U/q
Relation avec le champ : E⃗ = -∇V

Potentiel d'une charge ponctuelle : V = kq/r
Différence de potentiel : ΔV = V_B - V_A = -∫[A→B] E⃗·dl⃗

CHAPITRE 2 : CONDUCTEURS ET DIÉLECTRIQUES

2.1 CONDUCTEURS EN ÉQUILIBRE
Propriétés :
- Le champ électrique à l'intérieur est nul
- Le potentiel est constant
- La charge se répartit à la surface

2.2 CAPACITÉ
Définition : C = Q/V
Capacité d'un condensateur plan : C = ε₀A/d
Énergie stockée : U = ½CV² = ½QV

2.3 DIÉLECTRIQUES
Constante diélectrique : ε = εᵣε₀
Capacité avec diélectrique : C = εA/d
Polarisation : P⃗ = ε₀χE⃗

CHAPITRE 3 : MAGNÉTOSTATIQUE

3.1 CHAMP MAGNÉTIQUE
Force de Lorentz : F⃗ = q(v⃗ × B⃗)
Force sur un courant : F⃗ = I(l⃗ × B⃗)

3.2 LOI DE BIOT-SAVART
Champ magnétique créé par un courant :
dB⃗ = (μ₀/4π)(I dl⃗ × r̂)/r²

3.3 LOI D'AMPÈRE
∫B⃗·dl⃗ = μ₀I_enlacé
Applications :
- Fil rectiligne : B = μ₀I/(2πr)
- Solénoïde : B = μ₀nI
- Toroïde : B = μ₀NI/(2πr)

CHAPITRE 4 : INDUCTION ÉLECTROMAGNÉTIQUE

4.1 LOI DE FARADAY
Force électromotrice induite :
ε = -dΦ/dt
où Φ = ∫B⃗·dA⃗ est le flux magnétique

4.2 LOI DE LENZ
Le courant induit s'oppose à la variation du flux magnétique.

4.3 AUTO-INDUCTION
Coefficient d'auto-induction : L = Φ/I
Force électromotrice auto-induite : ε = -L(dI/dt)
Énergie stockée : U = ½LI²

CHAPITRE 5 : ÉQUATIONS DE MAXWELL

5.1 ÉQUATIONS FONDAMENTALES
1. ∇·E⃗ = ρ/ε₀ (loi de Gauss)
2. ∇·B⃗ = 0 (pas de monopôle magnétique)
3. ∇×E⃗ = -∂B⃗/∂t (loi de Faraday)
4. ∇×B⃗ = μ₀J⃗ + μ₀ε₀∂E⃗/∂t (loi d'Ampère-Maxwell)

5.2 ONDES ÉLECTROMAGNÉTIQUES
Équation d'onde : ∇²E⃗ = μ₀ε₀∂²E⃗/∂t²
Vitesse de propagation : c = 1/√(μ₀ε₀) = 3×10⁸ m/s

Relation entre E et B : E = cB
Vecteur de Poynting : S⃗ = (1/μ₀)(E⃗ × B⃗)

CHAPITRE 6 : CIRCUITS EN RÉGIME VARIABLE

6.1 CIRCUIT RLC SÉRIE
Équation différentielle : L(d²q/dt²) + R(dq/dt) + q/C = ε(t)

Solution en régime libre : q(t) = Q₀e^(-γt)cos(ωt + φ)
où γ = R/(2L) et ω = √(1/LC - γ²)

6.2 RÉSONANCE
Fréquence de résonance : ω₀ = 1/√(LC)
Facteur de qualité : Q = ω₀L/R = 1/(ω₀RC)

6.3 PUISSANCE EN RÉGIME SINUSOÏDAL
Puissance instantanée : p(t) = v(t)i(t)
Puissance moyenne : P = VI cos(φ)
Facteur de puissance : cos(φ)

EXERCICES CORRIGÉS

Exercice 1 : Deux charges q₁ = 2 μC et q₂ = -3 μC sont distantes de 10 cm. Calculer la force électrostatique.
Solution : F = k|q₁q₂|/r² = 9×10⁹ × 2×10⁻⁶ × 3×10⁻⁶/(0,1)² = 5,4 N

Exercice 2 : Un condensateur de 100 μF est chargé à 12 V. Calculer l'énergie stockée.
Solution : U = ½CV² = ½ × 100×10⁻⁶ × 12² = 7,2 mJ

Exercice 3 : Un fil parcouru par un courant de 5 A. Calculer le champ magnétique à 2 cm.
Solution : B = μ₀I/(2πr) = 4π×10⁻⁷ × 5/(2π×0,02) = 5×10⁻⁵ T

FORMULES UTILES

- Force de Coulomb : F = kq₁q₂/r²
- Champ électrique : E = kq/r²
- Potentiel : V = kq/r
- Capacité : C = Q/V = εA/d
- Force de Lorentz : F = qvB
- Loi de Biot-Savart : dB = (μ₀/4π)(I dl sin θ)/r²
- Loi de Faraday : ε = -dΦ/dt
- Auto-induction : ε = -L(dI/dt)
- Vitesse de la lumière : c = 1/√(μ₀ε₀)
