12. EXERCICES RÉSOLUS

Exercice 1 : Un participant apprend une liste de 20 mots. Après 24h, il se souvient de 12 mots. Calculer le pourcentage de rétention.

Solution :
Pourcentage de rétention = (Mots rappelés / Mots appris) × 100
Pourcentage de rétention = (12 / 20) × 100 = 60%

Exercice 2 : Dans une expérience, le groupe expérimental (n=30) a une moyenne de 85 et le groupe contrôle (n=30) a une moyenne de 75. L'écart-type est de 10. Calculer la taille d'effet de Cohen.

Solution :
d = (M1 - M2) / σ
d = (85 - 75) / 10 = 1,0 (effet large)

13. FORMULES IMPORTANTES

- Coefficient de corrélation : r = Σ[(X-X̄)(Y-Ȳ)] / √[Σ(X-X̄)²Σ(Y-Ȳ)²]
- Taille d'effet de Cohen : d = (M1 - M2) / σ
- Pourcentage de rétention : % = (Rappelé / Appris) × 100
- Seuil différentiel : ΔI / I = constante (loi de Weber)
- QI : QI = (Âge mental / Âge chronologique) × 100
